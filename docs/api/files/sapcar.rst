SAP CAR
~~~~~~~

Initial research about the file format was performed by Martin Gallo and published in `Deep-dive into SAP archive file
formats <https://www.coresecurity.com/corelabs-research/publications/deep-dive-sap-archive-file-formats>`_ at the
`Troopers 2016 Security Conference <https://www.troopers.de/troopers16/agenda/>`_. Additional research was performed
by `Hans-Christian Esperer <hc@hcesperer.org>`_ and published in the
`Hascar <https://github.com/VirtualForgeGmbH/hascar>`_ tool.
